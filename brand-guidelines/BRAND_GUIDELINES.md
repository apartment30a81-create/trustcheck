# TrustCheck — Brand Guidelines

## Brand Overview

**TrustCheck** is a privacy-first content authenticity verification tool. Users upload images/videos to check if they're real or AI-generated. The brand communicates trust, verification, transparency, and modern technology.

**Target Audience:** General public + journalists
**Platform:** PWA (Progressive Web App)

---

## Logo System

### Variant 1 — Lettermark (Primary Icon)
A stylized "T" integrated with a circular gauge/shield element. The T's crossbar and vertical stem form the core of the mark, while surrounding gauge arcs represent the Trust Meter — always showing measurable confidence. The small green checkmark accent inside the stem signals verified authenticity.

**Use:** App icon, favicon, avatar, navigation bar, small spaces.

### Variant 2 — Wordmark
"TrustCheck" in a clean sans-serif treatment. "Trust" in regular weight grounded by a teal underline bar suggesting stability. A green dot indicator above the letter 'k' signals the verification/check action.

**Use:** Headers, loading screens, where space allows horizontal text.

### Variant 3 — Combination Mark
Full horizontal lockup: T-shield icon + "TrustCheck" wordmark + "Verify with confidence" tagline. The icon sits on the left, wordmark on the right, tagline below. A teal accent bar underlines the full brand name.

**Use:** Website hero, login screens, marketing materials, official documents.

### Variant 4 — Horizontal Lockup (Simplified)
Condensed version of the combination mark without the tagline. Icon + "TrustCheck" with teal underline.

**Use:** Product UI headers, documentation, email signatures.

## Typography

**Primary Font:** Inter, SF Pro Display, or system sans-serif
**Secondary Font:** Manrope (headings), system-ui (body)

### Hierarchy
- **H1:** 48px / Bold / Primary Blue or Dark Text
- **H2:** 32px / Semibold / Dark Text
- **H3:** 24px / Semibold / Dark Text
- **Body:** 16px / Regular / Dark Text
- **Small:** 14px / Regular / Gray Text
- **Caption:** 12px / Regular / Gray Text

---

## Logo Usage Rules

### Clear Space
Minimum clear space = height of the "T" in the logo. No other elements should intrude into this space.

### Minimum Size
- **Lettermark:** 24px (digital), 0.25in (print)
- **Wordmark:** 120px wide (digital)
- **Combination:** 200px wide (digital)

### Don'ts
- ❌ Do not stretch or distort
- ❌ Do not change colors arbitrarily
- ❌ Do not add shadows or effects
- ❌ Do not place on busy backgrounds without sufficient contrast
- ❌ Do not rearrange elements

### Background Compatibility
- **Light backgrounds:** Use standard color variant
- **Dark backgrounds:** Use white/dark variant (inverted)
- **Photos:** Use white or dark variant with drop shadow if needed

## Application Examples

### Trust Meter UI
The circular gauge element from the logo extends directly into the UI's Trust Meter — a circular gauge showing 0-100% confidence score. The gauge uses the Primary Blue → Accent Green gradient for positive results, and Primary Blue → Accent Amber for medium-range results.

### Verification Badge
A simplified version of the lettermark (just the T + checkmark) serves as the verification badge shown on verified content results.
